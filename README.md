
# GILDE

This repository is a Figma-generated frontend bundle for GILDE.

Original project:
https://www.figma.com/design/DGn8BwiXjGv4oHvsSVz1K9/GILDE

## What is inside

- `src/` contains the original React + Vite frontend code exported from Figma.
- `index.html` is the Vite entry shell for the React app.
- `preview.html` is a standalone static HTML preview you can open directly in a browser without installing Node.js.

## Fastest way to view the layout

If you just want to see and edit the markup visually:

1. Open `preview.html` in your browser.
2. Edit `preview.html` and refresh the page.

This is the easiest way to practice HTML/CSS and understand the structure visually.

## Work with the original frontend project

If you want to work with the real React code from Figma:

1. Install Node.js.
2. Run `npm install`
3. Run `npm run dev`

After that, Vite will start a local development server and you will be able to view the original app in the browser.

### One-click launch

You can also double-click:

- `start-original-landing.bat`

This starts the original React landing page and opens it in the browser at:

- `http://127.0.0.1:5173`

## Recommendation

If your goal is to learn frontend and see immediate visual changes, start with `preview.html`.
If your goal is to continue the exported Figma project as a real app, work inside `src/app/components/LandingPage.tsx` and run the Vite project.
